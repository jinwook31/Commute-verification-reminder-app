# ⏰ 전문연 출퇴근 기록 리마인더 앱 (Commute Verification Reminder)

KAIST 전문연구요원(및 연구자)을 위한 **자동 출퇴근 기록 감지 및 누락 방지 리마인더 안드로이드 앱**입니다. 
연구에 몰두하다가 아차 하는 순간 출퇴근을 찍지 않아 발생하는 불상사를 방지하기 위해 개발되었습니다.

## ✨ 주요 기능 (Key Features)

* **📩 자동 출퇴근 감지 (KRP 알림 연동)**
  * `NotificationListenerService`를 활용하여 KRP 앱의 '출근처리', '퇴근처리' 알림을 백그라운드에서 실시간으로 감지하고 기기에 기록합니다.
* **🚨 맞춤형 누락 리마인더 알람**
  * 사용자가 지정한 시간(예: 출근 마감 전 08:50, 퇴근 기준 18:10)에 출퇴근 기록이 DB에 없다면 강력한 전체 화면 알람을 발생시킵니다.

---

## ⚙️ 필수 권한 설정 안내 (Troubleshooting)

이 앱은 백그라운드 알림 감지와 정확한 시간의 알람 발생이 핵심이므로, 설치 후 **반드시 아래의 권한들을 수동으로 허용**해 주어야 완벽하게 동작합니다.

### 1. 알림 접근 허용 (Notification Listener)
KRP 앱의 알림을 읽어오기 위해 필요합니다.
* 앱 최초 실행 시 뜨는 권한 요청 화면에서 **'알림 접근 허용'** 스위치를 켜주세요.

### 2. 정확한 알람 설정 권한 (Android 12+)
도즈(Doze) 모드 등 절전 상태에서도 정확한 아침 시간에 알람을 울리기 위해 필요합니다.
* 앱 최초 실행 시 **'알람 및 리마인더'** 권한을 허용해 주세요.

### 3. 다른 앱 위에 표시 (System Alert Window)
유튜브나 카카오톡 등을 사용 중일 때도 리마인더를 놓치지 않고 전체 화면으로 띄우기 위해 필요합니다.
* 최초 실행 시 안내되는 화면에서 **'다른 앱 위에 표시'** 권한을 허용해 주세요.

### 4. 🚨 시스템 알림 채널 강제 설정 (중요)
안드로이드 정책상 기기 설정에 따라 팝업 알림이 제한될 수 있습니다. **알람 소리는 나는데 화면 상단에 팝업이 뜨지 않는다면 아래 설정을 꼭 확인하세요.**
* 스마트폰 `설정` > `애플리케이션` > `앱 이름(인증 리마인더)` > `알림` 으로 이동합니다.
* 하단의 **'개인정보 보호 > 알림'** 탭으로 이동합니다.
* 알림 허용 기능을 활성화합니다.
